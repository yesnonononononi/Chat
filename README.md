# Chat
__Description__:__单体架构应用程序,全栈项目,以聊天为主要功能,支持私聊(支持消息状态展示),群聊,视频通话,系统消息广播,全局消息提示,配备后台管理系统__

__前端__: Vue3,TypeScript,pinia,tailwind,Socket.io,WebRTC(liveKit),element-plus

__后端__:springboot3(jdk21) RabbitMQ Redis Mybatis Redission MySQL SpringCache

## 消息安全
采用rabbitMQ
- 消息队列异步入库,保证消息的快速送达
- 生产者安全 生产者确认机制保证消息正确的投递
- 消费者安全 死信队列保证消息不丢失
## 连接安全
Socket.IO保证
- 定时心跳保证连接活跃
- 指数退避策略降低连接请求频率
- 本地ConcurrentMap作为一级缓存,Redis作为二级缓存,增加连接与消息发送效率
## 接口安全
- 自定义注解配合lua脚本保证接口幂等性
